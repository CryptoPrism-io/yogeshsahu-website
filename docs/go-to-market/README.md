# Go-To-Market Pack

This folder contains the core positioning and sales assets behind the site.

Files:

- `linkedin-profile-copy.md` - headline, about section, featured items, and service-page copy
- `outreach-templates.md` - cold email, LinkedIn DM, referral ask, and follow-up templates
- `cto-diagnostic-offer.md` - the canonical offer sheet in Markdown
- `cto-diagnostic-offer.html` - print-ready one-page offer sheet

Core proposition:

`Hands-On Chief Solutions Architect / Fractional CTO for AI, fintech, and data-heavy products. I scope, architect, code, and lead client-facing engagements from discovery through delivery.`
